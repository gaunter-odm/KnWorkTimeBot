## Команды

> **del** { день }.{ месяц } **del 1.1** - Удалит запись за **_1 Января_** текущего года

> **del** { день } **del 1** - Удалит запись за **_1 Января_** текущего месяца

>**/s** - Показать записи за текущий месяц

>**/d** - Удалить запись за сегодня
